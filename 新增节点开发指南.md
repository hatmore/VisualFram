## 快速开始

### 前提条件

1. Visual Studio 2019 已安装
2. 项目已正确配置（依赖库齐全）
3. 理解Qt信号槽机制
4. 基本的C++17语法

### 5分钟快速创建一个新节点

#### Step 1: 复制现有项目

最简单的方式是复制一个类似的插件项目，例如复制 `ImageShow` 或 `ResultStateJudgment`：

```bash
# 在 Plugin 目录下
复制 Plugin/ImageShow 为 Plugin/MyCustomNode
```

#### Step 2: 重命名项目文件

```
MyCustomNode/
├── MyCustomNode.vcxproj                (改名: 从 ImageShow.vcxproj)
├── MyCustomNode.vcxproj.filters
├── MyCustomNode.vcxproj.user
├── MyCustomNode/
│   ├── MyCustomNode.h                  (新建)
│   ├── MyCustomNodeInterface.h          (新建)
│   ├── MyCustomNodeInterface.cpp        (新建)
│   ├── MyCustomNodeNodeModel.h          (新建)
│   ├── MyCustomNodeNodeModel.cpp        (新建)
│   ├── PlugInformation.json             (修改插件信息)
│   └── ... [其他通用文件]
```

#### Step 3: 添加到主解决方案

1. 在VS中打开 `VisualFram2D.sln`
2. 右键解决方案 → 添加 → 现有项目
3. 选择 `MyCustomNode/MyCustomNode.vcxproj`

#### Step 4: 编译

```
Build → Build Solution (F7)
```

编译成功后，DLL会输出到 `Build/Debug(Release)/plugins/MyCustomNode.dll`

// ===== 关键: 核心算法实现 =====
int MyCustomNode::RunningTool()
{
    try {
        // 记录开始时间
        auto start_time = std::chrono::high_resolution_clock::now();

        // ===== 第1步: 获取输入数据 =====
        if (ptrInVMMat == nullptr) {
            emit SendNodeRuningStateSyncSig(0.0f, RUNNING_FAIL);
            return -1;
        }

        cv::Mat input_image = ptrInVMMat->vmMat;
        if (input_image.empty()) {
            emit SendNodeRuningStateSyncSig(0.0f, RUNNING_FAIL);
            return -1;
        }

        // ===== 第2步: 处理图像 =====
        cv::Mat output_image = input_image.clone();

        // 例如: 灰度化处理
        if (input_image.channels() == 3) {
            cv::cvtColor(input_image, output_image, cv::COLOR_BGR2GRAY);
        }

        // 例如: 应用参数
        if (functionParam.enableBlur) {
            cv::GaussianBlur(output_image, output_image,
                            cv::Size(functionParam.blurKernel,
                                   functionParam.blurKernel),
                            0);
        }

        // ===== 第3步: 输出结果 =====
        // 更新输入数据（用于链式处理）
        ptrInVMMat->vmMat = output_image;

        // ===== 第4步: 计算执行时间 =====
        auto end_time = std::chrono::high_resolution_clock::now();
        float elapsed_ms = std::chrono::duration<float, std::milli>(
            end_time - start_time
        ).count();

        // ===== 第5步: 发送成功信号 =====
        emit SendNodeRuningStateSyncSig(elapsed_ms, RUNNING_SUCESS);

        return 0;  // 成功
    }
    catch (const std::exception& e) {
        qDebug() << "执行错误:" << e.what();
        emit SendNodeRuningStateSyncSig(0.0f, RUNNING_FAIL);
        return -1;  // 失败
    }
}

// ===== 节点绑定成功槽函数 =====
void MyCustomNode::AcceptNodeBindingSucceedSlot(const QtNodes::PortIndex port_index)
{
    // 当输入端口接收到来自上游节点的数据时被调用

    if (port_index == 1) {
        // 端口1: 图像输入
        auto iter = mapCGObjectFlowInNodeAndName.find(port_index);
        if (iter != mapCGObjectFlowInNodeAndName.end()) {
            auto ptr = std::static_pointer_cast<BaseNodeData>(iter->second.ptrNodedata);
            auto vm_object = ptr->GetCGObjectNodeFlow<VMObject>();

            // 更新UI显示
            ui.qLineEditSource->setText("已连接: " + vm_object->nodeDataType.name);
        }
    }
}

// ===== 序列化 (保存参数) =====
virtual QJsonObject MyCustomNode::SerializeFunciton() override
{
    QJsonObject json;

    // 保存算法参数
    json["enableBlur"] = functionParam.enableBlur;
    json["blurKernel"] = functionParam.blurKernel;
    json["threshold"] = functionParam.threshold;

    // 保存UI参数
    json["screenNum"] = uiParam.screenNum;

    return json;
}

// ===== 反序列化 (加载参数) =====
virtual void MyCustomNode::ExserializeFunciton(const QJsonObject& json) override
{
    // 加载算法参数
    if (json.contains("enableBlur")) {
        functionParam.enableBlur = json["enableBlur"].toBool();
    }
    if (json.contains("blurKernel")) {
        functionParam.blurKernel = json["blurKernel"].toInt();
    }
    if (json.contains("threshold")) {
        functionParam.threshold = json["threshold"].toDouble();
    }

    // 加载UI参数
    if (json.contains("screenNum")) {
        uiParam.screenNum = json["screenNum"].toInt();
    }

    // 触发信号通知UI更新
    emit SendExserializeFinishSig();
}

// 在 MyCustomNode.cpp 的 RunningTool() 中

int MyCustomNode::RunningTool()
{
    auto start = std::chrono::high_resolution_clock::now();

    // 检查输入
    if (!ptrInVMMat || ptrInVMMat->vmMat.empty()) {
        emit SendNodeRuningStateSyncSig(0, RUNNING_FAIL);
        return -1;
    }

    cv::Mat src = ptrInVMMat->vmMat;
    cv::Mat dst;

    try {
        // 核心算法 (替换这部分)
        cv::cvtColor(src, dst, cv::COLOR_BGR2GRAY);

        // 保存结果
        ptrInVMMat->vmMat = dst;

        auto end = std::chrono::high_resolution_clock::now();
        float ms = std::chrono::duration<float, std::milli>(end - start).count();

        emit SendNodeRuningStateSyncSig(ms, RUNNING_SUCESS);
        return 0;
    }
    catch (const cv::Exception& e) {
        qDebug() << "CV Error:" << e.what();
        emit SendNodeRuningStateSyncSig(0, RUNNING_FAIL);
        return -1;
    }
}

int ThresholdNode::RunningTool()
{
    auto start = std::chrono::high_resolution_clock::now();

    try {
        if (!ptrInVMMat || ptrInVMMat->vmMat.empty()) {
            emit SendNodeRuningStateSyncSig(0, RUNNING_FAIL);
            return -1;
        }

        cv::Mat src = ptrInVMMat->vmMat;
        cv::Mat gray;

        // 如果是彩色，转灰度
        if (src.channels() == 3) {
            cv::cvtColor(src, gray, cv::COLOR_BGR2GRAY);
        } else {
            gray = src;
        }

        // 二值化
        cv::Mat dst;
        cv::threshold(gray, dst, functionParam.threshold,
                     functionParam.maxValue, functionParam.thresholdType);

        // 保存结果
        ptrInVMMat->vmMat = dst;

        auto end = std::chrono::high_resolution_clock::now();
        float ms = std::chrono::duration<float, std::milli>(end - start).count();

        emit SendNodeRuningStateSyncSig(ms, RUNNING_SUCESS);
        return 0;
    }
    catch (const cv::Exception& e) {
        qDebug() << "错误:" << e.what();
        emit SendNodeRuningStateSyncSig(0, RUNNING_FAIL);
        return -1;
    }
}

## 关键技术要点

### 1. 数据流向理解

```
┌─────────────────────────────────────┐
│  上游节点 A (ImageSource)            │
│  输出: cv::Mat 图像                  │
└────────────┬────────────────────────┘
             │ 通过 outData() 返回
             │ BaseNodeData::Ptr
             ▼
┌─────────────────────────────────────┐
│  中间节点 B (MyCustomNode)           │
│  接收: setInData() 调用              │
│  参数1: NodeData* 输入数据           │
│  参数2: PortIndex 输入端口号         │
└────────────┬────────────────────────┘
             │ Execute() / OnceExecute()
             │ 处理数据
             ▼
┌─────────────────────────────────────┐
│  下游节点 C (ImageShow)              │
│  接收: 处理结果                      │
└─────────────────────────────────────┘
```

## 常见问题解决

### Q1: 编译时找不到 Qt 头文件

**解决**:
```
项目属性 → VC++目录 → 包含目录
添加: $(QT_INSTALL_DIR)/include
```

### Q2: DLL 加载失败

**原因**:
- DLL 不在 `plugins/` 目录
- 依赖库缺失 (OpenCV, Qt DLL)
- 编译配置不匹配

**解决**:
1. 检查 DLL 输出位置
2. 复制所有依赖 DLL 到同一目录
3. 确保 Debug/Release 配置一致

### Q3: 节点不显示在工具树

**可能原因**:
- `PlugInformation.json` 配置错误
- `InterfacePlugin::ThisName()` 返回值错误
- `InterfacePlugin::NodeModelRegistry()` 未调用

**检查清单**:
```json
// PlugInformation.json 正确格式
{
    "IID": "Plugin.InterfacePlugin",
    "className": "MyCustomNodeInterface",
    "debug": false,
    "MetaData": []
}
```

### Q4: 输入端口无法接收上游数据

**原因**:
- 端口数据类型不匹配
- `InitialNodeData()` 未正确添加端口
- `mapCGObjectFlowInNodeAndName` 未复制到 NodeModel

**检查**:
```cpp
// 1. 确保添加了端口
mapCGObjectFlowInNodeAndName.insert(std::make_pair(1, in_node));

// 2. 确保 NodeModel 复制了映射
mapCGObjectFlowInNodeAndName =
    fram_dialog->mapCGObjectFlowInNodeAndName;

// 3. 检查数据类型一致
ptrInNodeVMMat->AddScopeNodeDataType(
    QtNodes::NodeDataType{ "5", "Mat" }  // 要与上游匹配
);
```

### Q5: 序列化后参数未恢复

**原因**:
- `ExserializeFunciton()` 实现不完整
- JSON key 名不一致
- 未发送 `SendExserializeFinishSig` 信号

**检查**:
```cpp
// 确保每个参数都被保存和恢复
virtual QJsonObject SerializeFunciton() override {
    QJsonObject json;
    json["param1"] = functionParam.param1;  // ✅ 保存
    json["param2"] = functionParam.param2;
    return json;
}

virtual void ExserializeFunciton(const QJsonObject& json) override {
    if (json.contains("param1")) {
        functionParam.param1 = json["param1"].toInt();  // ✅ 恢复
    }
    // ...
    emit SendExserializeFinishSig();  // ✅ 发送信号
}
```
